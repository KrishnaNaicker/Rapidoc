### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/[your-username]/rapidoc.git
   cd rapidoc
   ```

2. **Serve the website**
   ```bash
   # Option 1: Simple HTTP server

   ```

3. **Access the website**
   - Open `http://localhost:8000` in your browser
   - Or directly open `index.html` file
